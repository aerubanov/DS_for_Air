На графике ниже изображено измение концентрации частиц PM2.5 и PM10 и значений индекса AQI US (PM2.5) за последнее время и прогнозируемые значения. На графике AQU US цветом точек обозначен уровень опасности для здоровья (подробнее). На графике концентраций точки обозначают аномалии, а их цвет - принадлежность аномалии к кластеру. Для анализа используются средние по Москве значения с фильтрацией завышенных и заниженных значений.

Кластеризация аномалий

К кластеру 0 относятся аномалии, характеризующиеся небольшими значениями концентрации частиц . В кластере 2 увеличение PM2.5 связано с повышением влажности воздуха. В кластере 1 повышение концентрации частиц не связано с увеличением влажности. Ниже представлены гистограммы распределения признаков для различных кластеров

О проекте

Проект некомерческий и разрабатывается в рамках инициативы #ml4sg сообщества ODS.ai. Исходный код доступен на GitHub

Источники данных

В проекте используются проекта данные общественного мониторнга воздуха проекта Luftdaten. Данные собираются с датчиков концентрации частиц, температуры, атмосферного давления и влажности воздуха и размещаются в открытом доступе. В Москве установкой станций мониторинга качества воздуха занимаются волонтеры breathe.moscov.

Информация о погоде в Москве используется по данным метеостанции Балчуг
